Clase 06 - Unidades de Longitud y Modelo de Cajas

CSS tiene varias unidades diferentes para expresar una longitud como 10px, 2em, etc. Cuando se habla de longitud, esto se puede referir a: width, margin, padding, font-size, etc

El término "modelo de caja" se utiliza cuando se habla de diseño e interfaz (layout). Es esencialmente una caja que se ajusta a cada elemento HTML. Consiste en: márgenes, bordes, padding y el contenido real.

DIV

Content - El contenido de la caja, donde aparecen el texto y las imágenes • Padding - Limpia el área alrededor del contenido. Es transparente • Border - Un borde que rodea el padding y el content • Margin - Limpia el área fuera del borde. El margen es transparente • El modelo de caja nos permite agregar un borde alrededor de los elementos y definir el espacio entre los elementos.

SPAN

El modelo de caja nos permite agregar un borde alrededor de los elementos
